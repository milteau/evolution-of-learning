# 学习的进化

*用学习来理解学习，用进化来理解进化。*

---

你上一次真正改变自己学习方式，是什么时候？

不是换了一本新书，不是下载了一个新的笔记软件，而是从根本上重新思考——学习这件事，究竟是如何发生的？

大多数人的教育，教会了我们「学什么」，却几乎从不告诉我们「学习本身是什么」。我们被教导背诵、记忆、反复练习，却不知道这些动作背后的生物学逻辑。

这本小册子，是对这个问题的一次探索。
